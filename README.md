## 📁 About This Project

This repository contains the CFO investor dashboard built for the **BIOT 5219 Team Business Pitch Project**. The dashboard was designed in a **PitchBook-style** layout to support a simulated Series A investor presentation for **OncoClear Diagnostics** — a hypothetical multi-cancer early detection company using blood-based biomarker imaging.

---

## 📈 Dashboard Contents

| Chart | Description |
|-------|-------------|
| **KPI Strip** | TAM, SAM, SOM, test price, sensitivity, break-even year |
